How We Hunt
Distant Worlds

ExoHabitAI bridges the gap between raw NASA telemetry and a single interpretable Habitability Score, giving anyone the power to interrogate distant worlds.

// Why the Analysis Engine Dominates Noisy Astrophysics

Kepler photometry contains several noise sources: shot noise from photon counting, stellar variability (star spots, flares), and instrumental systematics. Classical models like SVM collapse under this correlated noise. The Prediction Model aggregates 500 decorrelated trees, each trained on a random bootstrap sample, producing stable low-variance estimates that are robust to these astrophysical artifacts.

500 Estimators
Ensemble averaging eliminates individual tree overfitting on Kepler measurement noise.
SMOTE Balancing
Synthetic minority oversampling applied to the rare habitable class counter dataset imbalance.
RFE Selection
Recursive feature elimination pruned 40+ to 22 high-signal predictors, cutting inference 38%.
GridSearchCV
Optimal: depth=None, n=500, min_samples_split=4. Tested 240 hyperparameter combinations.
